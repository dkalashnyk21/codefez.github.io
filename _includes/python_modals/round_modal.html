Round

Rounding is used to display a numerical value to it's nearest integer. For example 5.3 can be rounded to 5 or 3.333333333333 can be rounded
to 3.33 by default the round() method will
round to 2 decimal places unless specified. 

# Rounding to the nearest integer with no decimal places.
print(round(10/3,0)) 
# we can also use the int function to remove the decimal point.
print(int(round(10/3,0)))

# Rounding to 2 decimal places
print(round(10/3,2))
