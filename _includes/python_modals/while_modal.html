while

The while loop keeps running until a certain condition is met. You can replace the printing and incrementing with any code you want. 

                            
# The counter can help escape a loop
counter = 1

while counter <= 5:
    print(counter)

    # Increment the counter 
    counter += 1
                            
                        
1
    2
    3
    4
    5

If the counter is not outside of the while block, you may be stuck in a infinite loop.
